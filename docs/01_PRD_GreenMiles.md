## 1. 문서 개요

본 문서는 그린마일즈(GreenMiles) 프로젝트의 제품 요구사항을 사업적 관점에서 정의한다. 그린마일즈는 저탄소·로컬 농식품의 생산·유통·소비 데이터를 연결하여, 소비자의 장보기 행동을 탄소중립 참여로 전환하고, 유통사·식품기업에는 ESG 데이터를, 농가에는 추가 수익 기회를 제공하는 농식품 ESG 데이터 플랫폼이다.

본 문서는 개발팀, 디자이너, 사업 기획자, 투자자, 공모전 심사자가 함께 참고할 수 있도록 작성되었으며, MVP(최소 기능 제품) 범위를 중심으로 작성되었다. 본 서비스는 초기 단계에서 법정 탄소배출권을 발행하지 않으며, "자발적 탄소시장(VCM) 연계형 탄소 리워드 포인트"와 "예상 탄소 기여도"라는 표현을 사용한다.

|---|---|
| 프로젝트명 | 그린마일즈 GreenMiles |
| 서비스 성격 | 환경/기후테크, 농식품 ESG 데이터 플랫폼, 블록체인 기반 탄소 리워드 서비스, B2B SaaS + B2C 리워드 플랫폼 |
| 작성 단계 | MVP 기획 단계 |

## 2. 제품 비전

저탄소 농식품의 생산·유통·소비 데이터를 하나로 연결하여, **농가의 저탄소 실천이 수익으로 이어지고, 유통사의 ESG 대응이 데이터화되며, 소비자의 장보기 행동이 탄소중립 참여로 전환되는 시장 구조**를 만든다.

기존의 친환경 소비는 "좋은 행동"이라는 도덕적 동기에 머무르는 경우가 많았다. 그린마일즈는 이를 측정 가능하고 보상 가능한 경제적 구조로 전환하여, 저탄소 농식품 소비가 반복되고 확산되는 시장 메커니즘을 만드는 것을 목표로 한다.

## 3. 제품 목표

| 목표 | 설명 |
|---|---|
| 구매 데이터 수집 | 저탄소/로컬 농식품 구매 데이터를 POS 또는 QR/영수증 인증 방식으로 수집한다. |
| 탄소 기여도 산정 | 산지, 판매처, 운송거리, 저탄소 인증 여부, 생산 방식 정보를 결합해 탄소 기여도와 리워드를 산정한다. |
| 소비자 리워드 제공 | 소비자에게 GreenMiles 리워드 포인트를 제공해 즉각적인 혜택을 체감하게 한다. |
| 유통사 데이터 제공 | 유통사에 저탄소 상품 판매량, 리워드 발행량, 재구매율, 로컬푸드 전환율, Scope 3 참고 데이터를 제공한다. |
| 신뢰성 확보 | MRV/VCM 파트너 및 블록체인 기록 구조를 통해 향후 탄소 리워드의 신뢰성과 확장성을 확보한다. |

## 4. 문제 정의

| 이해관계자 | 문제/불편함 |
|---|---|
| 저탄소 농가/스마트팜 | 탄소 감축 노력을 수행해도 이를 추가 매출이나 가격 프리미엄으로 전환하기 어렵다. |
| 유통사/식품기업 | 저탄소 상품을 판매해도 상품별·고객별 탄소 데이터를 ESG 보고, Scope 3 관리, 친환경 CRM에 활용하기 어렵다. |
| 소비자 | 로컬푸드나 저탄소 농산물을 구매해도 즉각적인 혜택이나 자신의 탄소중립 기여도를 체감하기 어렵다. |
| 일반 개인 | 자발적 탄소시장(VCM)에 직접 참여하기에는 구조가 복잡하고 진입장벽이 높다. |
| 시장 전반 | 저탄소 농식품의 생산·유통·소비 데이터가 분절되어 있어, 농가·유통사·소비자·정책/금융기관이 같은 데이터를 활용하기 어렵다. |

## 5. 시장 배경

- ESG 공시 의무화 흐름과 함께, 식품·유통기업의 Scope 3(가치사슬 간접배출) 관리 필요성이 커지고 있다 `[AI 보완]`.
- 로컬푸드·저탄소 인증 농산물 시장은 존재하지만, 구매 시점에서 소비자에게 즉각적인 보상이 주어지는 구조는 드물다.
- 자발적 탄소시장(VCM)은 빠르게 성장하고 있으나 개인 소비자 단위 참여 채널은 제한적이며, 농식품 소비와 연계된 사례는 초기 단계다 `[추정]`.
- 유통사 입장에서는 친환경 마케팅보다 데이터 기반 CRM·재구매 전략에 활용 가능한 상품/고객 데이터에 대한 수요가 존재한다 `[AI 보완]`.

## 9. 이해관계자별 가치

### 9.1 소비자
- QR/영수증 인증만으로 구매를 즉시 인정받고 GreenMiles 포인트 적립
- 예상 탄소 기여도를 통해 자신의 친환경 소비를 체감
- 복잡한 VCM 개념 없이 "포인트 적립 + 혜택"으로 단순화된 경험

### 9.2 유통사
- 저탄소 상품 판매량, 리워드 발행량, 재구매율, 로컬푸드 전환율 데이터 확보
- Scope 3 참고 데이터로 ESG 보고 활용 가능
- 친환경 고객 세그먼트를 활용한 CRM/마케팅 가능성

### 9.3 저탄소 농가/스마트팜/로컬푸드 생산자
- 상품 등록만으로 소비자 판매 반응 데이터 확보
- 저탄소 실천이 정량 데이터로 기록되어 추후 가격 프리미엄·판로 확대 근거 자료로 활용 가능 `[추정]`

### 9.4 VCM/MRV 파트너
- 표준화된 구매·탄소 데이터를 통해 검증 대상 데이터를 쉽게 확보
- 향후 크레딧 연계 가능성 검토를 위한 데이터 파이프라인 확보 `[추정]`

### 9.5 정책/금융기관
- 농식품 탄소중립 관련 정책 설계·평가에 활용할 수 있는 실증 데이터 확보 `[AI 보완]`

## 12. MVP 제외 범위

- 대형마트 등 실제 POS 시스템과의 직접 연동
- 법정 탄소배출권의 실제 발행, 거래, 상쇄
- 블록체인 메인넷 배포
- 완전 자동화된 영수증 OCR 파이프라인 (MVP는 수동 승인 우선, OCR은 보조 기능)

- VCM 파트너와의 실제 크레딧 정산 계약 이행

## 13. 사용자 시나리오 예시

### 시나리오 1: 소비자 첫 구매 인증
1. 소비자가 회원가입 후 로컬푸드 매장에서 저탄소 인증 농산물을 구매한다.
2. 매장에 비치된 QR코드를 스캔하여 구매를 인증한다(또는 영수증을 업로드한다).
3. 시스템이 SKU·산지·판매처 데이터를 매칭하여 푸드 마일리지 기반 리워드를 계산한다.
4. 소비자는 지갑에서 적립된 GreenMiles 포인트와 예상 탄소 기여도를 확인한다.

### 시나리오 2: 유통사 대시보드 확인
1. 유통사 담당자가 매장과 SKU를 등록하고 가상 POS 구매 이벤트를 생성한다.
2. 한 달간 누적된 판매 데이터를 대시보드에서 확인한다.
3. 저탄소 상품 판매량, 리워드 발행량, 재구매율 데이터를 ESG 보고 자료로 활용한다.

### 시나리오 3: MRV 배치 처리
1. 관리자가 일정 기간 누적된 리워드 이벤트를 MRV 배치로 묶는다.
2. 배치 해시를 블록체인 테스트넷에 기록하여 데이터 위변조 가능성을 낮춘다.
3. 소비자 지갑에 "블록체인 기록 완료" 상태가 표시된다.

## 14. 비즈니스 모델

| 수익 모델 | 설명 |
|---|---|
| 유통사 SaaS 구독료 | ESG 대시보드, 리포트 Export 등 B2B 기능에 대한 월/연 구독료 |
| ESG/VCM 전환 컨설팅 | 유통사·식품기업 대상 Scope 3 데이터 활용, VCM 연계 컨설팅 |
| 리워드 거래/정산 수수료 | 향후 리워드 포인트가 실물 혜택이나 VCM 연계로 전환될 경우 발생하는 정산 수수료 `[추정]` |
| 친환경 고객 CRM/마케팅 데이터 활용 | 유통사가 친환경 고객 세그먼트 데이터를 마케팅에 활용할 수 있도록 제공하는 부가 데이터 상품 |

**왜 돈이 되는가:** 그린마일즈의 핵심 수익원은 B2C 리워드 자체가 아니라, 유통사·식품기업이 비용을 지불할 수 있는 **B2B SaaS형 ESG 데이터/대시보드**다. 소비자 리워드는 데이터 수집을 위한 참여 유인이며, 실제 수익화는 유통사의 구독료와 컨설팅, 데이터 활용에서 발생한다.

## 핵심 요약

그린마일즈는 저탄소/로컬 농식품 구매 데이터를 QR/영수증 인증으로 수집하고, 푸드 마일리지 기반 리워드를 소비자에게 제공하며, 유통사에는 ESG 데이터를 제공하는 B2B SaaS + B2C 리워드 플랫폼이다. MVP는 가상 POS, QR, 영수증 인증을 중심으로 작게 검증하며, 실제 수익화는 유통사 대상 SaaS 구독과 ESG 컨설팅에서 발생한다. 초기 단계에서는 법정 탄소배출권을 발행하지 않으며, 모든 리워드는 "예상 탄소 기여도"와 "GreenMiles 포인트"로 표현한다.
